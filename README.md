# tailwind table lists design
django-tailwind from paiza.cloud

```
git clone https://github.com/sin-sky-sora/firstproject.git
cd firstproject
sh init.sh
source ~/.bash_profile
pyenv install 3.8.8
pyenv global 3.8.8
# pipenv
python -m pip install --upgrade pip
python -m pip install pipenv
pipenv install
```

### start APP 
```
cd ~/firstproject/
pipenv run start

# tailwind css build
pipenv run build
# HTMLに使われてるCSSしかビルドされない!!!

```
